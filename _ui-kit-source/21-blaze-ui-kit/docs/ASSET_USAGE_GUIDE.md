
# Asset Usage Guide

- `home-lava-*`: Home, countdown, and dramatic result screens.
- `gameplay-embers*`: Gameplay and utility screens.
- `embers-overlay*`: Low-opacity decorative layer; never place above touch controls.
- `21-blaze-logo-*`: Main menu and marketing; use the smallest size that remains sharp.
- `countdown-fire-ring`: Use through `FireRingAnimation`.
- `fire-stopwatch-*`: Time-up result screen.
- SVG icons: editable master vectors and source for any custom icon pipeline.
- Audio: short original synthesized UI/game cues.
