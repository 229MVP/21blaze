
# UI QA Checklist

- [ ] All imports resolve.
- [ ] TypeScript syntax validates in the host app.
- [ ] No missing asset path.
- [ ] Transparent PNGs retain alpha.
- [ ] No watermark or outside branding.
- [ ] Cards are dynamic, not a 52-card sprite.
- [ ] Buttons show pressed/disabled/loading states.
- [ ] Modals have independent clickable actions.
- [ ] No nested Pressable controls.
- [ ] No horizontal overflow from 320–430 px.
- [ ] Safe areas are respected.
- [ ] Reduced Motion works.
- [ ] Screen reader labels describe cards and lanes.
- [ ] Gameplay logic remains outside UI components.
