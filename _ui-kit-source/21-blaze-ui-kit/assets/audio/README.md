
# Original Audio Placeholders

These short WAV files were synthesized specifically for this kit and contain no sampled third-party audio. They are intended as safe functional placeholders.

Recommended replacement targets:
- Button tap: 60–100 ms, dry click, no reverb.
- Card deal: 80–140 ms paper flick.
- Lane clear: 250–450 ms rising fire chime.
- Bust: 180–300 ms low impact/crackle.
- Countdown GO: 250–500 ms bright impact.
- High score: 500–900 ms celebratory fire fanfare.

Normalize conservatively and test on phone speakers.
