
# 21 Blaze UI Kit

Production-oriented Expo React Native UI kit for **21 Blaze**, based on the approved inferno arcade direction. The kit contains original raster/vector assets, dynamic playing cards, reusable TypeScript components, animation helpers, haptics, synthesized sound effects, ten component-based screen mockups, and integration documentation.

## Start here

1. Read `docs/INSTALLATION.md`.
2. Copy `assets/` and `src/` into your Expo project.
3. Install the packages listed in `package-additions.json`.
4. Register `BlazeUIKitPreviewScreen` only in development or preview builds.
5. Integrate components into existing screens without replacing gameplay logic.

No font files are included. Install the documented open-license font packages through npm/Expo.
