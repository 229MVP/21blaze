# Feature Freeze — Progression Beta 0.5B

Progression Beta 0.5B is the **final new feature milestone** before Production 1.0.

No major new features may be added before Production 1.0.

## Permitted work before Production 1.0

Only these categories are allowed:

- Critical bug fixes
- Security fixes
- Performance fixes
- Accessibility fixes
- Store compliance
- Analytics corrections
- Purchase reliability
- Ad reliability
- Backend reliability
- UI polish that does not alter scope

## Deferred post-launch features

The following are intentionally deferred and must not ship in this freeze window:

- Premium Season Pass
- Blaze Club subscription
- Tournaments
- Spectator mode
- Clans
- Creator events
- Rematches
- Advanced replays
- Additional game modes
