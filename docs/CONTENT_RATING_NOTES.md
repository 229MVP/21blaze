# Content Rating Notes — 21 Blaze RC 0.9.0

Guidance for filling IARC / App Store / Google Play content questionnaires.  
Final ratings must be generated by the store questionnaires; this document is engineering’s honest content description.

---

## What 21 Blaze is

Fast-paced digital card / number game (“21” / Blaze lanes) with:

- Solo scoring runs
- Optional online leaderboards
- Optional live casual and ranked multiplayer
- Cosmetic themes (cards/arenas)
- Optional ads and IAPs (remove ads, cosmetics, Founders, Pro subscriptions)

---

## Questionnaire expectations (draft)

| Topic | Suggested answer | Rationale |
|-------|------------------|-----------|
| Cartoon / fantasy violence | None / no | No combat depiction |
| Realistic violence | No | |
| Sexual content / nudity | No | |
| Profanity | No (confirm UI copy) | Audit UI strings before submit |
| Drugs / alcohol / tobacco | No | |
| Horror / fear | No | Inferno/volcano cosmetics are stylistic, not horror gameplay |
| Gambling — real money | **No** | No cash-out, no wagering real currency |
| Simulated gambling | **Usually No** | Score-based card play; cosmetics named “Neon Casino” are **cosmetic only** — do not award real-world prizes. If a questionnaire asks about casino imagery, disclose cosmetic theme only |
| User-generated content | Limited — display names | Provide reporting/block plan if required |
| Unrestricted web access | No | |
| Location sharing | No (confirm SDKs) | |
| Advertising | **Yes** (unless ads removed build) | Interstitial + rewarded via AdMob |
| In-app purchases | **Yes** | Cosmetics, remove ads, Founders, Pro |
| Shares personal info | Account identifiers / gameplay with backend | See privacy map |
| Age directed primarily under 13 | **No** (confirm product intent) | COPPA implications if changed |

---

## Monetization honesty for ratings

- Ads currently configured with **Google test IDs** — still disclose ads intent for production.
- Pro subscriptions exist (monthly/yearly/lifetime) — declare subscriptions.
- Purchases do **not** alter match fairness / card RNG (stated in product copy) — keep true in practice.

---

## Multiplayer

- Players may see others’ display names and scores.
- No voice/text chat claimed.
- Two-device behavior **untested** — does not change content rating, but does block quality sign-off.

---

## Screenshots / store art constraints

Avoid implying:

- Real-money gambling or payouts
- “Casino winnings”
- That neon/volcano arenas affect odds

Prefer: skill, speed, streaks, cosmetics, ranked competition.
