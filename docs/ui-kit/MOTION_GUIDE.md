
# Motion Guide

- Button press: 90–140 ms scale.
- Card deal: 180–240 ms fade/translate.
- Lane clear: 300 ms pulse plus game-owned particles.
- Bust: 220 ms horizontal shake.
- Score count: 400–700 ms depending on delta.
- Countdown: 700–900 ms per beat.
- Final seconds: controlled pulse; never block input.
- Ember drift: decorative only; stop when the screen loses focus.

Respect system Reduced Motion and the game setting. Essential state changes must remain visible without animation.
