
# Design System

## Direction
Dark premium arcade UI with controlled orange/gold fire accents. Functional information remains code-rendered and high contrast. Decorative artwork is always separable from controls.

## Tokens
- Background: near-black, blue-black, charcoal.
- Fire: ember red, orange, bright orange, gold, pale flame.
- Text: warm white, cool gray, muted blue-gray.
- Borders: subtle gray, dim orange, active gold, danger red.

Use exports from `src/theme`. Avoid raw colors in screen code.

## Layout
- Mobile-first: 320–430 px.
- Standard panel radius: 10–14 px.
- Standard gap: 8–16 px.
- Primary button: 48–56 px minimum height.
- Gameplay lanes: 2×2 grid with stable equal widths.
