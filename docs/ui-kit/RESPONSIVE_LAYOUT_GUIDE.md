
# Responsive Layout Guide

Test at 320, 375, 390, 430 px, tablet, and web.

- Use safe-area insets for headers and bottom actions.
- Permit the stat row to wrap to 2×2 below roughly 340 px if labels clip.
- Keep lane cards tiny and overlapping; never shrink text below readable sizes.
- Avoid screenshot-specific absolute positioning except decorative overlays.
- Use `aspectRatio` for cards and ring artwork.
- Limit content width on tablet/web while keeping backgrounds full bleed.
